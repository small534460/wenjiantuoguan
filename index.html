使用说明 (点击展开) ▼
  • 1. 获取 GitHub Token：访问 GitHub → Settings → Developer settings → Personal access tokens → Generate new token
  • 2. 权限设置：需要勾选 repo (完全控制仓库) 权限
  • 3. 保存 Token：在上方输入框中粘贴 Token 并点击"保存 Token"
  • 4. 加载文件：点击"加载文件"按钮查看仓库中的文件
  • 5. 上传文件：选择文件并点击"上传文件"按钮
  • 6. 下载文件：点击文件名旁边的"下载"按钮
  • 7. 删除文件：点击文件名旁边的"删除"按钮（需要确认）
  • 注意：你的文件会发送到small 3的仓库保管
