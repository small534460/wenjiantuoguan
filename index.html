GitHub 配置

如何使用：

  • 1. 在 GitHub Token设置页面 创建一个新的个人访问令牌
  • 2. 选择 repo 权限（完全控制私有和公共仓库）
  • 3. 将生成的令牌粘贴到上面的Token输入框中
  • 4. 填写您的GitHub用户名和仓库名称
  • 5. 点击"保存配置"保存设置，然后点击"加载文件"查看仓库中的文件
